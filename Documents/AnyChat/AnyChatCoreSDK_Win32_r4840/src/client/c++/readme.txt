BRAnyChatSDKDemo	标准的C++示例代码，展示了AnyChat SDK的基本工作流程
			参考：http://www.anychat.cn/faq/index.php?action=artikel&cat=2&id=193&artlang=zh

AnyChatInputVideo	演示“外部音视频数据输入”接口的使用。将外部的音频数据、视频数据替换为本地视频采集
			参考：http://www.anychat.cn/faq/index.php?action=artikel&cat=2&id=254&artlang=zh


AnyChatVideoMixer	将指定房间的所有用户视频进行合成，音频进行混音，变成一路视频流，一路音频流，当其它用户请求该合成视频流时，可以显示多窗口画面，使用了“外部音视频数据输入”接口，适合于大型会议使用。
