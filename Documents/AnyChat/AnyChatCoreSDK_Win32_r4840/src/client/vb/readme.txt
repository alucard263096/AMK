
一、编译环境：VB.Net（VS2008）
	编译运行时，需要将"bin\client\BRAnyChatCore.dll"文件拷贝到bin\x86\Debug目录，否则会提示加载dll失败

二、使用说明
1、拷贝AnyChat相关组件（bin\client\*.dll）到应用程序目录下；
2、启动程序（Mini_Demo\bin\x86\Release\Mini_Demo.exe）；
3、输入您要登录的用户名和进入的房间编号，点击登录按钮执行登录操作；
4、成功登陆系统后，系统进入主界面，并自动打开本地视频和音频；
5、勾选小喇叭的前的复选框来打开（或关闭）本地音频；
6、拖动耳机图标和麦克风图标所示的滑动条可以分别控制本地音频输出和输入的音量大小；
7、房间用户列表显示在最左边的列表框中，您可以通过双击某个房间用户名来请求该用户的视频和音频。远程视频将显示在右上角的窗口中；
8、中间部分的是信息输出窗口和文字消息输入文本框。请在下部的文字消息输入文本框输入您要发送的文字，点击发送按钮发送文字消息; 
9、程序运行过程中，将会在当前目录下产生日志文件“BRAnyChatCore.log”，程序工作不正常时，可查看日志了解详细情况；
10、该程序仅用于演示AnyChat Platform Core SDK在VB.Net环境中的工作流程，有关SDK相关的API接口调用请参考相关文档或访问“AnyChat知识中心”网站：http://www.anychat.cn/faq/ ，在使用过程中遇到任何疑问，请与我们联系。
